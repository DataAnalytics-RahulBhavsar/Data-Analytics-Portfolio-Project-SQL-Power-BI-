# ☕📊 Data Analytics Portfolio Project (SQL + Power BI)

## 🔍 Project Overview

This project is a **complete end-to-end data analytics workflow** where raw (unclean) data is transformed into meaningful business insights using **SQL** and **Power BI**.

The project demonstrates the full data lifecycle:
➡️ Raw Data → Data Cleaning → Data Analysis → Data Visualization → Business Insights

It includes multiple datasets and dashboards to showcase practical, real-world data analysis skills.

## 🧠 Key Insights (Summary)

* 📌 Sales show **seasonal growth patterns**, especially in later months
* 📌 A small number of products contribute to a large portion of revenue
* 📌 Some products/customers generate **low profit despite high sales**
* 📌 Presence of "Unknown" categories indicates **data quality issues**
* 📌 Certain regions and locations need **performance improvement strategies**
